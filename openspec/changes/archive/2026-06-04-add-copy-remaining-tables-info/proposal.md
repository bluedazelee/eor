## Why

賽事進行中，承辦人員常需向主辦或其他裁判回報「目前還剩哪些桌沒處理完」，尤其要區分加時桌與一般桌。目前只能人工檢視圖卡逐一抄寫，費時且容易出錯。需要一鍵將剩餘桌次資訊整理成可分享的文字並複製到剪貼簿。

## What Changes

- 主畫面新增「複製剩餘桌次資訊」按鈕
- 點擊後依未完成桌次（狀態非「已完成」）產生資訊字串，並複製到剪貼簿
- 字串格式：`剩餘 n 桌；剩餘加時桌：{未完成加時桌清單}；剩餘一般桌：{未完成一般桌清單}`
  - 加時桌顯示桌號與加時分鐘數，例如 `12(+3分)`
  - 一般桌僅顯示桌號
  - 兩段桌號皆由小到大排序
  - 某段無資料時顯示「無」
- 複製成功/失敗給予使用者回饋

## Capabilities

### New Capabilities
- `copy-remaining-tables-info`: 依未完成桌次產生剩餘桌次資訊字串並複製到剪貼簿

### Modified Capabilities
（無）

## Impact

- `index.html`：新增「複製剩餘桌次資訊」按鈕的 HTML 元素
- `app.js`：新增按鈕 DOM 參考、字串組裝邏輯與 Clipboard API 複製與回饋
- `style.css`：新增按鈕樣式（若需要）

# PTCG EoR Tracker

供 PTCG（寶可夢集換式卡牌遊戲）賽事裁判使用的 **End of Round 桌次管理工具**。以 PWA 形式運作，支援離線使用。

---

## 功能說明

### 一、開局設定

開啟工具後在設定畫面填入以下欄位，按「開始紀錄」：

| 欄位 | 說明 |
|------|------|
| 目前輪次 | 輸入輪次名稱，例如 `R1`、`R2`；下一輪會自動遞增數字 |
| 組別 | 從下拉選單選擇組別（大師 A~G 組、少年 A~B 組、孩童 A~B 組），上次選擇的組別會被記住 |
| 起始桌號 | 負責範圍的起始桌 |
| 結束桌號 | 負責範圍的結束桌 |

---

### 二、桌次狀態管理

每張桌次卡片有三種狀態，**單點卡片**可循環切換：

| 狀態 | 顯示 | 說明 |
|------|------|------|
| `尚未結束`（紅色） | 預設初始狀態 | 比賽仍在進行 |
| `已分配`（橘色） | 顯示分配時間（HH:MM） | 點擊後記錄當前時間，表示裁判已前往處理 |
| `已完成`（綠色） | — | 該桌次處理完畢 |

> 切換順序：尚未結束 → 已分配 → 已完成 → 尚未結束

---

### 三、加時桌標記

**操作：** 長按（按住 0.5 秒）桌次卡片，開啟加時設定彈窗

- 快速選擇：**+3 / +6 / +9 分鐘**
- 自訂輸入：填入任意正整數分鐘數，點「確認」
- 清除標記：若已設定加時，彈窗會顯示「清除標記」按鈕
- 點彈窗外區域或右上角 ✕ 可關閉

加時桌會在卡片上顯示橘色 `+N分` 徽章。

---

### 四、儀表板統計

追蹤畫面頂部即時顯示：

- 已完成桌數 / 總桌數（百分比）
- 剩餘桌數
- 進度條視覺化

---

### 五、篩選與顯示控制

| 控制項 | 說明 |
|--------|------|
| 「顯示已完成桌號」勾選框 | 取消勾選可隱藏已完成的桌次，專注於未完成部分 |
| 「只顯示加時桌」按鈕 | 切換為只顯示有加時標記的桌次（可與「顯示已完成」複合過濾） |

---

### 六、複製資訊

點「複製資訊」按鈕，展開選單，有三種複製格式：

| 選項 | 複製內容範例 |
|------|-------------|
| **本輪範圍** | `大師A組 R1 1~50` |
| **加時桌次** | `12(+6分)`<br>`23(+3分)` |
| **詳細資訊** | 包含本輪範圍、剩餘桌數、加時桌清單、一般未完成桌清單 |

---

### 七、結束本輪

| 按鈕 | 說明 |
|------|------|
| **確認本輪處理完畢** | 所有桌次皆為「已完成」時才可點擊，確認後清除紀錄並進入下一輪設定 |
| **強制結束本輪** | 無論完成狀態，強制結束並清除紀錄，下一輪設定前會顯示仍有幾桌未完成 |

---

### 八、資料持久化與 PWA

- 所有狀態即時儲存至 **localStorage**，重新整理或關閉視窗後仍保留，下次開啟自動恢復追蹤畫面
- 支援 **PWA 安裝**（可加到主畫面），支援**完全離線使用**（Service Worker 快取策略）
- 頁面右上角顯示目前是「在線」或「離線模式」

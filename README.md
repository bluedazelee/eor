# PTCG EoR Tracker

供 PTCG（寶可夢集換式卡牌遊戲）賽事裁判使用的 **End of Round 桌次管理工具**。以 PWA 形式運作，支援離線使用。

---

## 功能說明

### 一、開局設定

開啟工具後在設定畫面填入以下欄位，按「開始紀錄」：

| 欄位 | 說明 |
|------|------|
| 目前輪次 | 輸入輪次編號（1–20），下一輪會自動遞增；顯示格式為 R1、R2… |
| 組別 | 自由輸入組別名稱（必填），上次輸入的內容會被記住 |
| 起始桌號 | 負責範圍的起始桌 |
| 結束桌號 | 負責範圍的結束桌 |

---

### 二、桌次狀態管理

每張桌次卡片有三種狀態，**單點卡片**可循環切換：

| 狀態 | 顯示 | 說明 |
|------|------|------|
| `尚未結束`（紅色） | 預設初始狀態 | 比賽仍在進行 |
| `已分配`（橘色） | 顯示分配時間（HH:MM） | 點擊後記錄當前時間，表示裁判已前往處理 |
| `已完成`（綠色） | — | 該桌次處理完畢 |

> 切換順序：尚未結束 → 已分配 → 已完成 → 尚未結束

---

### 三、加時桌標記

**操作：** 長按（按住 0.5 秒）桌次卡片，開啟加時設定彈窗

- 快速選擇：**+3 / +6 / +9 分鐘**
- 自訂輸入：填入任意正整數分鐘數，點「確認」
- 清除標記：若已設定加時，彈窗會顯示「清除標記」按鈕
- 點彈窗外區域或右上角 ✕ 可關閉

加時桌會在卡片上顯示 `+N分` 徽章，顏色依加時長度分為三個等級：

| 等級 | 條件 | 顏色 |
|------|------|------|
| 低 | ≤ 3 分 | 深金黃 |
| 中 | 4–6 分 | 深琥珀橘 |
| 高 | > 6 分 | 紅色 |

---

### 四、儀表板統計

追蹤畫面頂部即時顯示：

- 已完成桌數 / 總桌數（百分比）
- 剩餘桌數
- 進度條視覺化

---

### 五、篩選與顯示控制

| 控制項 | 說明 |
|--------|------|
| **隱藏完成** | 隱藏已完成桌次，專注於未完成部分；再次點擊恢復顯示 |
| **只顯示加時** | 只顯示有加時標記的桌次，可與「隱藏完成」複合使用 |
| **+ / −** | 調整每列桌次數量：`+` 增加每列桌次（3 桌 → 4 桌 → 5 桌），`−` 減少每列桌次；偏好設定會自動記住 |

#### 範圍篩選標籤

當桌次總數超過 25 桌時，卡片上方會自動出現範圍分段標籤。點擊標籤可快速聚焦至特定桌次區段，括號內數字為該區段的未完成桌數；點擊「全部」恢復顯示全部桌次。範圍篩選可與其他篩選條件複合使用。

---

### 六、桌號搜尋

點擊按鈕列最右側 🔍 按鈕，開啟數字鍵盤彈窗。點按數字鍵後即時跳轉至對應桌次並高亮顯示；按 ⌫ 可退格，點「關閉」或彈窗外區域可關閉。

- 若桌號不存在，彈窗顯示「找不到桌號 X」
- 若目標桌次被篩選器隱藏（隱藏完成、只顯示加時、範圍標籤），系統自動清除相關篩選並直接跳轉，彈窗顯示「已清除篩選，跳轉至桌號 X」
- 關閉彈窗後高亮持續 1 秒再消失；重新開啟彈窗時立即清除

---

### 七、批量設為已完成

點擊「批量完成」按鈕進入選取模式：

| 操作 | 說明 |
|------|------|
| **點選卡片** | 切換選取狀態（綠框 ✓ 表示已選取），不影響桌次完成狀態 |
| **全選** | 一鍵選取目前顯示中所有尚未完成的桌次 |
| **清除** | 取消所有已選取的桌次 |
| **反選** | 將目前顯示的桌次中，已選者取消、未選者加入 |
| **確認** | 將所有已選取的桌次標記為已完成 |
| **取消** | 退出選取模式 |

批量模式中可使用「隱藏」與「加時」篩選器，搭配「全選」快速選取特定條件的桌次；也可使用範圍篩選標籤聚焦至特定區段後再執行全選。

---

### 八、複製資訊

點擊 📋 按鈕展開選單，有四種複製格式：

| 選項 | 複製內容 |
|------|---------|
| **本輪範圍** | 組別、輪次與桌號範圍，例：`大師A R1 1~50` |
| **加時桌次** | 各加時桌的桌號與加時分鐘數 |
| **剩餘桌次** | 組別、輪次、桌號範圍及剩餘桌數 |
| **詳細資訊** | 完整摘要：範圍、剩餘桌數、加時桌與一般桌清單 |

---

### 九、復原與重做

| 按鈕 | 說明 |
|------|------|
| **↩ 復原** | 還原到上一次操作前的狀態，支援最多 20 步；批量操作亦可一步還原 |
| **↪ 重做** | 復原後可重新套用被復原的操作；執行任何新操作後重做紀錄清除 |

按鈕呈半透明時表示目前無可復原／重做的步驟。重新整理頁面後歷史記錄清除，但桌次狀態仍保留。

---

### 十、結束本輪

底部按鈕會依完成狀態自動切換：

| 狀態 | 按鈕 | 說明 |
|------|------|------|
| 仍有未完成桌次 | **強制結束本輪**（琥珀色） | 無論完成狀態均可點擊，確認後清除紀錄並準備下一輪 |
| 所有桌次已完成 | **確認本輪處理完畢**（綠色） | 自動切換，點擊確認後清除紀錄並準備下一輪 |

---

### 十一、資料持久化與 PWA

- 所有狀態即時儲存至 **localStorage**，重新整理或關閉視窗後仍保留，下次開啟自動恢復追蹤畫面
- 支援 **PWA 安裝**（可加到主畫面），支援**完全離線使用**（Service Worker 快取策略）
- 頁面右上角顯示目前是「在線」或「離線模式」
